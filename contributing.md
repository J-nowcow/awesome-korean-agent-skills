# 기여 가이드

이 프로젝트에 기여해주셔서 감사합니다! 한국어 AI 코딩 에이전트 스킬 생태계를 함께 만들어갑시다.

## 항목 추가 기준

추가하려는 항목이 다음 조건을 충족하는지 확인해주세요:

- **한국어 지원**: 스킬/룰 자체가 한국어이거나, README에 한국어 문서가 있거나, 한국어 자동 감지를 지원
- **동작 확인**: 직접 설치하고 사용해본 항목만 추가 (링크만 보고 추가하지 않기)
- **유지 관리**: 최근 6개월 이내에 업데이트된 레포 우선
- **설명 포함**: 왜 이 스킬이 유용한지 한 줄로 설명

## PR 절차

1. 이 레포를 Fork합니다
2. 적절한 카테고리에 항목을 추가합니다
3. 다음 포맷을 따릅니다:

```markdown
| 스킬명 | 유형 | 도구 | 설명 | 언어 | 레포 |
```

- **유형**: 🔧 Skill / 🤖 Agent / ⚡ Command / 🪝 Hook
- **도구**: CC (Claude Code) / GC (Gemini CLI) / CX (Codex CLI) / CR (Cursor) / OC (OpenCode)
- **언어**: `한국어` / `영+한` / `다국어(KO)`

4. PR을 생성합니다
5. PR 설명에 해당 스킬을 직접 사용해본 경험을 간략히 적어주세요

## 새 카테고리 제안

기존 카테고리에 맞지 않는 스킬이 있다면 Issue를 먼저 생성해주세요.

## SKILL.md 작성 템플릿

직접 만든 스킬을 공유하고 싶다면, 별도 레포에 아래 형식으로 만든 뒤 이 목록에 추가해주세요:

```yaml
---
name: skill-name
description: 스킬 기능 한 줄 설명
---

# 스킬 이름

## 사용 시기
- 사용 사례 1
- 사용 사례 2

## 지침
[에이전트용 상세 지침]

## 예제
[실제 사용 예제]
```
