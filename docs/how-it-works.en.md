# 🤖 How Does This Repo Work?

> awesome-korean-agent-skills is **100% maintained by AI agents**.

## Automation Structure

4 GitHub Actions workflows:
- 🔍 skill-scout (every Monday): Auto-discover new skills + classify + open PR
- 🔗 link-checker (daily): Detect dead links + auto-remove
- ⭐ weekly-picks (every Monday): Auto-rotate "Skill of the Week"
- 📊 sync-counts (daily): Sync category item counts + date

## How It Works

### 1. Skill Discovery (skill-scout)
- Collects candidates via GitHub Search API (topics, SKILL.md filenames, keywords)
- Deduplicates against already-registered repos (data/known-repos.json)
- Uses Google Gemini API to verify Korean language support + classify category
- Opens a PR → waits 24 hours → auto-merges

### 2. Link Validation (link-checker)
- Sends HTTP HEAD requests to all URLs daily
- 404 / timeout → auto-removed

### 3. Weekly Picks (weekly-picks)
- Criteria: diversity, practicality, Korean language support quality
- Recommendation reasons auto-generated by Gemini API
- Prevents duplicates with previous picks

### 4. Count Sync (sync-counts)
- Updates README item counts and date daily

## Data Files
| File | Purpose |
|------|---------|
| data/known-repos.json | Index of processed repos |
| data/weekly-picks-history.json | History of past picks |

## The Human's Role
- Supervision: review auto-generated PRs
- Intervention: close PR if classification is wrong
- Category management: humans decide new categories (agents suggest via Issues)

## Want to Build Your Own?
1. Reference workflow files in .github/workflows/
2. Clone scripts in .github/scripts/
3. Set GEMINI_API_KEY Secret
4. Modify scripts to match your category structure
